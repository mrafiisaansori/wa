Kirim Pesan

Pesan masuk antrian dan dikirim dengan jeda otomatis (anti-blasting).

Format: kode negara tanpa + atau spasi, mis. 62812xxxxxxx

Keterangan Status

Arti tiap status di tabel riwayat.

  • Antri - pesan diterima, menunggu giliran kirim (ada jeda antar pesan).
  • Terkirim - sudah diteruskan ke server WhatsApp.
  • Delivered - sudah sampai di HP penerima.
  • Read - pesan sudah dibaca penerima.
  • Gagal - pengiriman gagal (nomor tidak valid, gateway belum terhubung, dll).
